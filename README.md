# 星芒起始页 (Star.Lumina Start Page)

## 项目简介
星芒起始页是一个简洁美观的浏览器起始页，提供快速搜索、常用网站导航和个性化设置功能。

## 主要功能
- **智能搜索**：支持多种搜索引擎快速切换
- **网站导航**：分类整理常用网站，一键直达
- **主题设置**：支持浅色/深色/自动主题切换
- **背景设置**：必应每日一图或自定义背景
- **个性化**：可自定义链接打开方式等设置

## 技术特点
- 响应式设计，适配各种设备
- 基于localStorage的个性化设置保存
- 毛玻璃效果设计
- 动态主题切换

## 使用方法
1. 将本项目部署到Web服务器
2. 在浏览器设置中将此页面设为起始页
3. 在设置页面中自定义您的偏好

# Star.Lumina Start Page

## Project Introduction
Star.Lumina Start Page is a clean and beautiful browser start page that provides quick search, frequently used website navigation and personalized settings.

## Main Features
- **Smart Search**: Support multiple search engines with quick switching
- **Website Navigation**: Categorized collection of frequently used websites
- **Theme Settings**: Support light/dark/auto theme switching
- **Background Settings**: Bing daily picture or custom background
- **Personalization**: Customizable link opening methods and other settings

## Technical Features
- Responsive design, adapts to various devices
- Personalized settings saved via localStorage
- Glassmorphism design
- Dynamic theme switching

## Usage
1. Deploy this project to a web server
2. Set this page as the start page in browser settings
3. Customize your preferences in the settings page
